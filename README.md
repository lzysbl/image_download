##版本说明
当前版本有windows完整版代码，下载器纯净代码，和可运行exe文件
## 使用方式

- **exe文件可以直接执行**
    
    可以直接在windows环境下运行
- **编译代码运行**

    用编译器执行需要安装必要环境依赖，执行以下命令
    ```
    pip install -r requirements.txt
    ```
